# AppVentasInventario

## Descripción
Aplicación de escritorio diseñada para el control operativo de inventario y registro de ventas. El sistema integra un módulo de análisis predictivo que utiliza inteligencia artificial para proyectar la demanda futura de productos.

## Funcionalidades Principales
* **Registro de Ventas:** Gestión transaccional en tiempo real.
* **Control de Inventario:** Administración de stock, entradas y salidas de mercancía.
* **Predicción de Demanda:** Módulo especializado en proyecciones de inventario para optimizar el reabastecimiento.

## Tecnologías
* **Interfaz de Usuario:** JavaFX
* **Lenguaje Principal:** Java
* **Análisis Predictivo:** Python (Librería Prophet)
* **Base de Datos:** SQLite

## Requisitos del Sistema
* Java Development Kit (JDK) 17 o superior.
* Python 3.8 o superior con las siguientes librerías:
  * `prophet`
  * `pandas`
